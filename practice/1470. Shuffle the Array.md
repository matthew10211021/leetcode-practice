## 1470. Shuffle the Array
### Algorithm
- Create a new array of size `2n`
- Use a two pointer technique to fill the elements in the array
### Implementation
Python Solution:
```python
def shuffle(self, nums, n):
    result = [0] * (2 * n)
    l, r = 0, n
    flag = 1  # 1: left; -1: right

    for i in range(2 * n):
        if flag == 1:
            result[i] = nums[l]
            l += 1
        else:
            result[i] = nums[r]
            r += 1
        flag *= -1  # switch the flag

    return result
```
C++ Solution:
```cpp
vector<int> shuffle(vector<int>& nums, int n) {
    vector<int> result(2 * n, 0);
    int l = 0, r = n;
    int flag = 1;

    for (int i = 0; i < 2 * n; ++i) {
        if (flag == 1) {
            result[i] = nums[l];
            ++l;
        }
        else {
            result[i] = nums[r];
            ++r;
        }
        flag *= -1;  // swap
    }

    return result;
}
```
